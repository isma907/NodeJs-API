# NodeJS + Express + MongoDB + TypeScript
# Characters API
# 📦 Project Setup

## 1. Install dependencies

Make sure you have the following installed:

- Node.js
- MongoDB

---

## 2. Configure database

Start your MongoDB service locally or use a remote instance.

---

## 3. Seed the database

Run the following command to populate initial data:

```bash
npm run seeds
```

## 4. Run Project

```bash
npm start
```
